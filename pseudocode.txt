Pseudocode

1. PRINT mode menu
2. LET input operation mode 
3. CHECK whether the input is an integer in [1,2,3,4], each number corresponds to an operation- if yes proceed to next, if not ask the user to re-enter an input
4. LET input first number
5. LET input second number
6. CHECK if the inputs are integers- if yes, proceed to the next, if not ask the user to re-enter the inputs
7. CHECK if the divisor is non-zero (for division)- if yes proceed to next, if not ask the user to re-enter inputs
7. DO the arithmetic operation
9. PRINT the result
10. LET input exit -if yes break the loop, if not continue the loop
